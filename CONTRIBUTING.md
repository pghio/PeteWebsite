# Contributing

This repository is Peter Ghiorse's public portfolio and writing system. Small fixes to accessibility, broken links, build behavior, or factual transcription are welcome.

Before opening a pull request:

1. Create a focused branch.
2. Run `npm ci` and `npm run check`.
3. Explain the user-visible change and how it was verified.
4. Do not add private prompts, personal visitor data, credentials, or employer-confidential information.

Editorial submissions and unsolicited rewrites are out of scope. If a published claim is wrong, open an issue with the source and the narrow correction proposed.
