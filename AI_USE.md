# AI use and accountability

AI tools support implementation, research retrieval, adversarial review, data checks, and editing in this repository. Pete Ghiorse remains accountable for the product decisions, source selection, evaluation design, factual claims, code accepted into the repository, and final published language.

## Working standard

- Claims must be traceable to a dated source, a reproducible public artifact, or a clearly labeled private case study.
- Generated code is reviewed, tested, and committed by the accountable maintainer.
- AI-generated summaries do not replace inspection of the underlying data or source.
- Material limitations and corrections remain visible beside the work they qualify.
- Private prompts, conversations, credentials, customer information, and employer-confidential material are not published.

Older commit metadata may identify an AI coding tool as author. History is not being rewritten. Current work uses Pete as the accountable author and describes material AI assistance in the pull request or repository documentation.
